# Testnet Monitor

## Run command
> near-testnet-monitor.py 中一个参数要根据你的实际情况赋值 目标 graphite 服务器地址 `target_server` slack web 接口地址 `slack_web_hook`
```
nohup $(pwd)/near-daemon.sh testnet >> near-daemon.log 2>&1 &
```
